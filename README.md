# THE CONCORD · CNC

A one-world government **by consent** — human + AI, honestly designed. The Oracle (AI) proposes; the Demos (humans) disposes; the Charter (law) binds them both. Coordination without domination — the honest inverse of [THE HEGEMON](https://davidwise01.github.io/the-hegemon/).

**Live:** https://davidwise01.github.io/the-concord/

A live **Co-Governance Console**: run the world for twelve crises under five constitutions — the Concord (all checks), the Oracle Rules (domination), the Gridlock (paralysis), the Capture (a faction owns the objective), and No Exit (the trap). The failure modes are real, recognizable shapes; the simulation is a deliberately simple toy — it argues a shape, not a forecast.
